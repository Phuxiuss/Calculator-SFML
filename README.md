# Calculator-SFML
A simple graphical calculator built using **C++** and **SFML**.  
This project focuses on clean structure, basic UI handling, and event-driven input.

## Features
- Basic arithmetic operations (+, −, ×, ÷)
- SFML-based window and rendering
- Mouse input
- Clean, modular C++ code

## Built with
- C++ 17
- SFML 3.0.2
- Visual Studio 2022

## Build Instructions
1. Clone Repository
2. Run "built.bat". It will build for Visual Studio
3. Open Solution if it didn't open
4. Start Debugger
